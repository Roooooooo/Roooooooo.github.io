# 标识符

区分大小写/首字符必须是字母、下划线或$/惯例使用驼峰大小写形式，首字母小写后面每个字母大写，非强制。

# 语句

以分号结尾（推荐，非强制），优点：有助于解析器确定语句结尾，防止输入内容不完整，提升性能，便于压缩代码，删除空格。

代码块，以{}包裹，在控制语句中使用能让内容更清晰，减少出错概率。

# 变量

ECMAScript是松散类型变量，变量可以保存任何类型的数据，有3个关键字可以声明变量var(所有版本)、let、const(ES6)。

var：全局作用域声明的var变量会自动挂载到window对象上。声明范围为函数作用域。声明提升，使用var声明的变量会自动提升到函数作用域的顶部。

    console.log(age); //undefined
    var age = 26;

等价于

    var age;
    console.log(age);//undefined
    var age = 26;

let：声明范围为块作用域。没有声明提升，在声明之前使用会“暂时性死区”，抛出“ReferenceError”。全局作用域声明的变量不会挂载到window对象上。

const：声明时必须初始化，不允许修改（限制只适用于它指向的变量引用），如果声明的是一个对象，可以修改这个对象的内部属性。其他同let。

    const name = "rong";
    name = "guo";//TypeError:给常量赋值
    const person = {};
    person.name = "rong";

最佳实践：const优先，let次之，不使用var。

# 数据类型

ECMAScript有6种简单数据类型（原始类型）：Undefined、Null、Boolean、Number、String、Symbol(ES6新增)，1种复杂数据类型Object。通过typeof操作符可以确定任意变量的数据类型，会返回如下字符串之一：
"undefined": 值未定义；
"boolean"：布尔值；
"string"：字符串；
"number"：数值；
"object"：null或者对象；
"function"：函数；
"symbol"：符号；
