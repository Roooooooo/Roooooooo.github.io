# JavaScript的前世今生

# JavaScript历史

1.1995年，JavaScript1.0诞生，作者：网景公司Brendan Eich，目的：为客户端提供简单的数据验证。

2.1996年，网景在Netscape Navigator3中发布了JavaScript1.1版本，同年微软发布了包含JavaScript实现的JScript的IE3，重磅进入了Web浏览器领域。

3.1997年，JavaScript1.1提交到欧洲计算机制造商协会，同年EMCAScript作为JavaScript脚本语言标准发布。

4.1998年，国际标准化组织（ISO）和国际电工委员会（IEC）也就ECMAScript采纳为标准，从此以后，各个浏览器均以ECMAScript作为JavaScript的实现依据。

# JavaScript与ECMAScript

ECMAScript是JavaScript的实现依据，JavaScript包含但不限于ECMAScript。
完整的JavaScript包含：核心（ECMAScript）、文档对象模型（DOM）、浏览器对象模型（BOM）。

# ECMAScript版本

第1版：基本上通JavaScript1.1，删除了所有浏览器特定代码，外加少量细微的修改。

第2版：无增减和更改任何特性。

第3版：更新字符串处理、错误定义、和数值输出，增加了正则表达式、新的控制语句、try/catch异常处理的支持。

第4版（废弃）：包含强类型变量、新语句和数据结构、真正的类和继承。

第5版：因第4版跳跃太大遭废弃，委员会将3.1版本作为第5版发布，增加原生的解析和序列化JSON数据的JSON对象，增加严格模式。

第6版：俗称ES6、ES2015，于2015年6月发布，正式支持了类、模块、迭代器、生成器、箭头函数、promise、反射、代理和众多新的数据类型。

第7版：于2016年6月发布，少量增强。

第8版：于2017年6月发布，增加async/await异步函数等。

第9版：于2018年6月发布，少量增强。

第10版：于2019年6月发布，少量增强。
